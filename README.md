# Emirates-USA-Operational-Overview-2024-
Built in **Power BI**, this project simulates an Emirates framework using **341,000+ flight records**. Using **Power Query** for ETL and masking, it tracks an **81.08% On-Time Performance (OTP)**. The dashboard leverages **DAX** to map a mid-year low of **66.12%** across **65,000 delayed flights**.
